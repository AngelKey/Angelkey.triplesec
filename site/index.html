TripleSec

Governments have a lot of power. Consider:

  • the USA may compel websites & businesses to "cooperate"
  • the NSA may have already broken a popular cipher

If you're a programmer and wish to store user data that only the user can read -- to the exclusion of both you and the government -- you must encrypt it in the browser or client app with extreme presumption. With privacy in mind, TripleSec is a simple, open-source symmetric encryption library in JavaScript. It hides data using AES, Salsa 20, and Twofish, so that a failure in one will expose nothing. The 3 encryptions are combined in a simple technique outlined by Bruce Schneier here.

In other words, TripleSec should work, unless all 3 of these encryptions have fallen.
